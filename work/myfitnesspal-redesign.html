About
Conducted a usability test in which a participant was asked to perform various tasks on the MyFitnessPal website, including finding subscription information, setting up an account, and browsing through exercises. This test revealed some issues with MyFitnessPal's overall usability. I have redesigned five different screens on MyFitnessPal's website based on the findings of the usability test.
Premium Section on Homepage
  • Addresses the issue of not being able to view the price of a premium plan or information about it on the homepage
  • Added anavigable “Premium” button to the header
  • Included available price points and features included in premium page
  • User can click on “Learn more” to go to the full premium page
Creating a Usermane
Original
Redesign
  • Addresses the issue of it being difficult to create a unique username during the sign up process
  • Gives unique suggestions based on the user's name
  • Suggestions can be refreshed for new options
Option to Choose Premium Plan after Signing Up
Original
Redesign
  • Addresses issue after the sign up process which leads the user to pay for premium account without a clear way to start using the free account
  • Clearly gives two options: a free plan and a premium plan
  • Each plan displays the features and a button to use the service with the corresponding plan
Entering Exercise Information
Original
Redesign
  • Addresses the issue of the exercise input section not specifying time or distance, or showing a unit
  • Each input section clearly specifies what it’s asking for, and displays the unit, which the user setsy
  • Moved the effort of activity (leisure, light, moderate, etc.) to after choosing an exercise
