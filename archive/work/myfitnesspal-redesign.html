What is MyFitnessPal?
MyFitnessPal is a popular mobile app and website that helps users track their diet and exercise, featuring a food diary for logging meals, an exercise tracker for monitoring physical activities, and nutritional information to help users achieve their health and fitness goals.
About the Project
Conducted a usability test in which a participant was asked to perform various tasks on the MyFitnessPal website, including finding subscription information, setting up an account, and browsing through exercises. This test revealed some issues with MyFitnessPal's overall usability, which are outlined below.
  • No way to go back to the homepage from the signup/onboarding page
  • Wasn’t clear that you could scroll down more on the homepage
  • Potentially unfamiliar jargon (pescatarian)
  • Base measurement units should be the common measurement units of the country accessing the site
  • Difficult to create a username that is not already in use
  • Forces user to pay for premium account after signing up without a clear way to start using the free account
    • Not much information about what the premium subscription offers before paying
  • Lack of visual affects/stimulation in sign up process
  • Need specific input for adding an exercise
    • Entering “Bike ride” into the search bar did not yield results for a bike activity, needed to only type “bike”
  • “How long” for exercise input section does not specify time or distance, or shows a unit
I also reviewed the usability of the site myself and found a few more issues:
  • No clear way to access primary content of app without signing up
    • Need to scroll to the bottom and in small text
    • When navigating to the food, exercise, or apps page from the bottom and go back home, those pages are no longer easily accessible at the top
  • Buttons to toggle between the reviews and stories section are small and hard to click
  • “Start today” and “take the quiz” lead to same sign up process, which can be misleading
  • “About MyFitnessPal” section is very small and hard to read at the bottom of the homepage
I have redesigned five different screens on MyFitnessPal's website based on the findings of the usability test.
Premium Section on Homepage
  • Addresses the issue of not being able to view the price of a premium plan or information about it on the homepage
  • Added anavigable “Premium” button to the header
  • Included available price points and features included in premium page
  • User can click on “Learn more” to go to the full premium page
Creating a Usermane
Original
Redesign
  • Addresses the issue of it being difficult to create a unique username during the sign up process
  • Gives unique suggestions based on the user's name
  • Suggestions can be refreshed for new options
Option to Choose Premium Plan after Signing Up
Original
Redesign
  • Addresses issue after the sign up process which leads the user to pay for premium account without a clear way to start using the free account
  • Clearly gives two options: a free plan and a premium plan
  • Each plan displays the features and a button to use the service with the corresponding plan
Entering Exercise Information
Original
Redesign
  • Addresses the issue of the exercise input section not specifying time or distance, or showing a unit
  • Each input section clearly specifies what it’s asking for, and displays the unit, which the user setsy
  • Moved the effort of activity (leisure, light, moderate, etc.) to after choosing an exercise
